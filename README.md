# docker-homer

Serving [bastienwirtz/homer](https://github.com/bastienwirtz/homer) in a scratch image. Intended to be ran in a Kubernetes environment.  
Some resources pre-gzipped. Image Size ~1.07MB. Memory usage: ~304KB.

The artifact generated by this repo assumes the container image is mounted as an image-volume to a pod running a http webserver. For example:

```yaml
spec:
  containers:
    - name: homer
      image: ghcr.io/trexx/docker-busybox-httpd:1.37.1-1
      volumeMounts:
        - mountPath: /etc/homer/
          name: config
        - name: www
          mountPath: /www
      ports:
      - name: http
        containerPort: 8080
  volumes:
    - name: config
      configMap:
        name: config
    - name: www
      image:
        reference: ghcr.io/trexx/docker-homer:v25.11.1-2
```
